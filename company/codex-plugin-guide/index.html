Company Guide

Codex에서 공식 플러그인 설치 및 사용하기

Codex 앱에서 제공하는 공식 플러그인을 설치하고, 플러그인 안에 포함된 Skill을 실제 작업에 활용하는 방법입니다.

Install

공식 플러그인 설치 흐름

Step 1

좌측 메뉴에서 플러그인 열기

  • Codex 앱을 실행합니다.
  • 좌측 사이드바에서 플러그인을 클릭합니다.
  • 플러그인과 Skill을 관리하는 화면으로 이동합니다.
Step 2

플러그인 목록 확인하기

  • 상단에서 플러그인 탭이 선택되어 있는지 확인합니다.
  • OpenAI 엄선 영역에서 공식 플러그인을 확인합니다.
  • 이미 연결된 플러그인과 추가 가능한 플러그인을 구분합니다.
  • 필요한 플러그인이 보이지 않으면 검색창을 사용합니다.
Step 3

원하는 플러그인 검색하기

  • 검색창에 설치하려는 플러그인 이름을 입력합니다.
  • 예시: superpowers
  • 검색 결과에서 원하는 플러그인을 확인합니다.
  • 오른쪽의 연결 버튼을 클릭합니다.
Step 4

설치 완료 확인하기

  • 상단에 Superpowers 플러그인을 추가했습니다 메시지가 표시되는지 확인합니다.
  • 플러그인 항목 오른쪽에 체크 표시가 보이는지 확인합니다.
  • 아래에 함께 표시되는 Skill 목록을 확인합니다.
  • 설치 후 새 채팅을 열고 플러그인을 사용합니다.
Useful Skills

이 플러그인의 유용한 Skill 사용방법

1

Writing Plans

바로 작업을 시작하기 전에, 목표를 실행 가능한 계획으로 나누고 확인 기준을 정리할 때 사용합니다.

  • 큰 작업을 단계별 계획으로 쪼갤 때
  • 작업 전에 누락된 요구사항을 확인할 때
  • 완료 기준과 검증 방법을 먼저 정할 때
2

Brainstorming

아이디어를 바로 확정하지 않고, 여러 방향으로 넓혀본 뒤 더 좋은 선택지를 고를 때 사용합니다.

  • 기능 아이디어를 여러 개 뽑아볼 때
  • 디자인/콘텐츠 방향을 비교할 때
  • 막연한 요청을 구체적인 선택지로 바꿀 때
Examples

Skill 요청 예시

Skill을 직접 지정해서 사용하려면 채팅창에 $를 입력한 뒤 원하는 Skill을 선택하거나, $brain, $writing Plans처럼 Skill 이름을 함께 입력합니다.

Writing Plans 예시. 교육자료 제작 계획 만들기

Superpowers의 Writing Plans를 사용해서, 비개발자 대상 “AI 플러그인 활용 기초 교육” 자료를 만드는 계획을 세워줘.

교육 시간은 45분이고, 대상자는 Codex를 처음 써보는 구성원이야.

목차 구성, 필요한 예시 화면, 실습 여부, 준비물, 발표자가 확인해야 할 체크리스트까지 포함해서 계획을 먼저 작성해줘.

Brainstorming 예시. 교육자료 개선 아이디어 넓히기

Superpowers의 Brainstorming을 사용해서, 비개발자 대상 “AI 플러그인 활용 기초 교육”을 더 쉽게 만드는 아이디어를 여러 방향으로 뽑아줘.

대상자는 Codex를 처음 써보는 구성원이고, 가장 궁금해하는 것은 공식 플러그인을 설치하고 실제 업무에 적용하는 방법이야.

바로 하나의 결론으로 정하지 말고, 교육 흐름 개선, 실습 아이디어, 예시 화면 구성, 난이도 낮추는 방법, 후속 교육 주제까지 나눠서 아이디어를 제안해줘.
Tips

사용할 때 기억할 점

플러그인 설치 후에는 새 채팅에서 사용하는 것이 안전합니다. 일부 플러그인은 추가 로그인이나 권한 연결이 필요할 수 있습니다. 회사 계정 정책에 따라 사용할 수 있는 플러그인이 다를 수 있습니다.
Extra Guide

oh-my-codex 설치방법

1. README를 읽고 설치 요청하기

  • 새 Codex 채팅을 엽니다.
  • GitHub 링크와 함께 README를 읽고 설치해달라고 요청합니다.
  • Codex가 README의 설치 조건과 필요한 명령을 먼저 확인하게 합니다.
  • 권한 승인이 필요하면 설명을 확인한 뒤 승인합니다.
GitHub README 열기 →

2. Omx Setup 진행하기

  • 설치 후에는 Omx Setup을 진행합니다.
  • Codex에게 Omx Setup 진행해줘라고 요청합니다.
  • setup 과정은 시간이 조금 소요될 수 있습니다.
  • setup 과정에서 질문이 나오면 순서대로 성실히 답변합니다.
  • setup이 끝나면 정상 설치 여부와 사용 가능한 명령을 확인합니다.
Deep Interview

oh-my-codex에서 자주 쓰는 Skill 예시

1

Deep Interview

요청이 아직 애매할 때, 바로 작업하지 않고 질문을 통해 목적과 범위를 먼저 정리할 때 사용합니다.

  • 무엇을 만들어야 할지 아직 명확하지 않을 때
  • 대상자, 범위, 완료 기준을 먼저 잡아야 할 때
  • AI가 섣불리 실행하지 않고 필요한 질문부터 하게 만들 때

Deep Interview 예시. 요구사항을 먼저 깊게 정리하기

$deep-interview
사내 구성원들이 Codex에서 공식 플러그인을 설치하고 실제 업무에 적용할 수 있도록, 따라하기 쉬운 가이드를 만들고 싶어.

내가 아직 정확히 정하지 못한 부분도 있을 수 있으니, 바로 만들지 말고 필요한 질문을 하면서 목적, 대상자, 범위, 예시, 최종 결과물 기준을 먼저 정리해줘.
